🔨 実装手順（簡略版）

Step 1: 電源ラインの配線

基板の上端に赤線で+9Vライン、下端に黒線でGNDラインを配線します。これが全体の電源供給の基準になります。

Step 2: 真空管ソケットの取付

9ピンソケットを基板中央に配置し、各ピンをはんだ付けします。ピン1の位置を確認してください。

Step 3: 受動部品の取付

抵抗とコンデンサーを図の位置に配置します。電解コンデンサーは極性に注意してください。

Step 4: 信号配線

入力→グリッド→プレート→出力の順に信号経路を配線します。ノイズ対策のため、配線は短く保ちます。

Step 5: スイッチとジャックの接続

SW1、SW2、入出力ジャック、ボリュームポットをリード線で接続します。シールド線の使用を推奨します。

✅ 動作確認チェックリスト

  • □ DCジャック極性確認（センターがマイナス、外側がプラス）
  • □ 電源ラインのショートチェック（+9VとGND間）
  • □ ヒーター電圧確認（Pin4とPin5の間で約9V DC）
  • □ 各スイッチの導通確認
  • □ 入出力ジャックの配線確認
  • □ LED極性の確認（長い足がアノード）
  • □ LED抵抗（1kΩ）が直列に入っているか確認
  • □ スターグランド配線の確認（すべて入力ジャックに集中）
  • □ 真空管挿入後、ヒーターの発光確認
  • □ 電源LED（赤）の点灯確認
  • □ SW2切替時の青LED動作確認
  • □ 音出しテスト（ボリューム最小から開始）
  • □ 4つの音色モードすべての動作確認
🔥 安全上の注意：
  • はんだ付け作業は換気の良い場所で行ってください
  • 真空管は動作中熱くなります。触れないよう注意してください
  • 電源投入前に必ず配線をダブルチェックしてください
  • DCジャックの極性を必ずテスターで確認（BOSSはセンターマイナス）

💡 LED抵抗値の計算ガイド

LED保護抵抗の選定

LED順電圧（Vf）と必要な電流から抵抗値を計算します。

計算式

R = (電源電圧 - LED順電圧) ÷ LED電流
R = (9V - Vf) ÷ I_LED

実例

LED色 典型的Vf 推奨電流 計算値 使用抵抗
赤 2.0V 10mA 700Ω 1kΩ
青 3.3V 10mA 570Ω 1kΩ
超高輝度 - 5mA - 2.2kΩ

※ 1kΩで眩しすぎる場合は2.2kΩに変更

【準備編】必要な工具と材料

🤔 なぜこれらの工具が必要なのか？

電子工作は精密作業です。適切な工具を使うことで、作業効率が上がるだけでなく、部品を傷めず、安全に作業できます。特に初心者の方は、良い工具を使うことで失敗を減らせます。

工具

  • はんだごて（20-30W、温度調節可能なら350℃設定）
    → 真空管ソケットは熱に弱いため、適切な温度管理が重要
  • はんだ（0.8mm径、ヤニ入り）
    → 細めのはんだは温度管理しやすく、きれいな仕上がりに
  • ニッパー（リード線カット用）
    → 切れ味の良いものを使うと、部品にストレスをかけません
  • ワイヤーストリッパー（被覆剥き用）
    → カッターよりも確実に、芯線を傷つけずに剥けます
  • ピンセット（細かい作業用）
    → 小さな部品の位置決めや、熱い部品の取り扱いに必須
  • テスター（導通・電圧確認用）
    → 目に見えない電気の流れを確認する「目」の役割
  • フラックス（必要に応じて）
    → はんだの乗りが悪い時の救世主
  • はんだ吸い取り線（修正用）
    → 失敗しても修正できる安心感が大切

配線材

  • 単線（0.5-0.6mm径）各色：
    → 基板上の配線に使用。単線は形が保持されて作業しやすい
    • 赤（電源+）、黒（GND）：電源系統を明確に
    • 白（入力）、黄（出力）：信号の流れを視覚化
    • 青、灰：スイッチ配線で機能を区別
    • 緑：ヒーター専用で他と区別
    • 橙：内部接続の識別用
  • より線（フレキシブル配線用）：
    → 可動部分や外部接続に使用。柔軟性があり断線しにくい
  • シールド線（推奨）：
    → 入出力配線に使うとノイズを大幅に減らせます

推奨する追加購入品

  • 熱収縮チューブ（各サイズ）
    → 接続部の保護とショート防止に必須。プロの仕上がりに
  • ICクリップ（テスター用）
    → 動作確認時に基板を触らずに測定できて便利
  • LEDベゼル（3mm用×2個）
    → ケースにLEDをきれいに取り付けるため
  • ケーブルタイ
    → 配線の整理整頓に

【工程1】基板の準備と電源バスラインの作成

🤔 なぜこの作業をするのか？

電源バスラインは、電車の線路のような役割を果たします。すべての部品に安定した電源を供給するための「共通の道」を最初に作ることで、後の配線が簡単になり、電源ノイズも減少します。建築で言えば、水道管と下水管を最初に通すようなものです。

作業内容

  1. ユニバーサル基板（30×20穴）を用意し、表面の汚れを除去
    → 汚れがあるとはんだが乗りにくくなるため
  2. 基板の向きを決定（部品面が上、はんだ面が下）
    → 統一することで配線ミスを防ぐ
  3. GNDライン作成：黒線を基板下端（Y=18の位置）に沿って配線
    • 始点：左端から3穴目（X=3, Y=18）
    • 終点：右端から3穴目（X=27, Y=18）
    • 単線を使用し、各10穴ごとにはんだ付けで固定
    → GND（グランド）は電気の帰り道。すべての電流はここに戻ってきます
  4. +9Vライン作成：赤線を基板上端（Y=2の位置）に沿って配線
    • 始点：左端から3穴目（X=3, Y=2）
    • 終点：右端から3穴目（X=27, Y=2）
    • GNDラインと同様に固定
    → +9Vは電源の供給源。上下に離すことで誤接続を防ぎます

確認ポイント

✓ テスターで+9VラインとGNDライン間がショートしていないことを確認（抵抗値＝∞）

→ ショートしていると電源を入れた瞬間に大電流が流れて危険です

【工程2】真空管ソケットの取付

🤔 なぜこの作業をするのか？

真空管ソケットは、この回路の心臓部です。真空管は高価で繊細な部品なので、直接はんだ付けせず、ソケットを使って着脱可能にします。また、真空管は消耗品なので、交換できることが重要です。中央に配置するのは、すべての配線を最短距離で結べるためです。

作業内容

  1. 9ピンソケット（B9A）の向きを確認
    • ピン1とピン9の間が広い間隔になっている
    • 基板中央（X=15, Y=10）にソケット中心を配置
    → ピン番号を間違えると回路が正しく動作しません
  2. ソケットのピンを基板の穴に差し込む
    • 無理に押し込まず、穴位置を確認しながら挿入
    • ソケットが基板に密着するまで押し込む
    → 浮いていると振動で接触不良を起こします
  3. 裏面からはんだ付け
    • まず対角の2ピン（Pin1とPin5など）を仮固定
    • ソケットの水平を確認してから全ピンをはんだ付け
    • はんだ付け時間は各ピン3秒以内（ソケットの熱損傷防止）
    → プラスチック製ソケットは熱に弱く、長時間加熱すると変形します

確認ポイント

✓ 各ピンが確実にはんだ付けされているか目視確認

✓ 隣接ピン間のショートがないかテスターで確認

→ ピン間のショートは真空管を破損させる可能性があります

【工程3】入力部の配線（C1とグリッド接続）

🤔 なぜこの作業をするのか？

C1（カップリングコンデンサ）は、ギターからの音声信号（AC）は通すけれど、直流（DC）は通さない「ゲート」の役割をします。これにより、前段の機器のDC成分が真空管のグリッドに影響を与えることを防ぎます。0.022μFという小さい値は、低音をカットして、ギターアンプらしい明るい音色を作ります。Rg（グリッドリーク抵抗）は、グリッドに適切なDC基準電位を与える重要な部品です。

作業内容

  1. C1（0.022μF）の取付
    • 位置：真空管ソケットの左側（X=8, Y=9）から（X=8, Y=11）
    • リード線を適切な長さにカットして成形
    • 部品面から挿入し、裏面ではんだ付け
    → フィルムコンデンサは極性がないので、どちら向きでもOK
  2. Rg（1MΩ グリッドリーク抵抗）の取付【重要】
    • 位置：（X=8, Y=12）から（X=8, Y=14）垂直配置
    • カラーコード確認：茶・黒・緑・金
    • 上端をC1出力側と同じポイントに接続
    • 下端をGNDラインへ
    → この抵抗がないとグリッドが「浮いて」動作不安定になります！
  3. 入力ジャック配線の準備
    • 白色のより線を20cm程度用意
    • 両端の被覆を5mm剥く
    • 片端を基板の入力パッド（C1の左側）に予備はんだ
    → より線を使うのは、ジャックへの配線が曲げやすく断線しにくいため
  4. C1出力側からPin2（グリッド）への配線
    • 白色の単線で（X=8, Y=11）から（X=13, Y=11）へ配線
    • Pin2の位置を確認して確実に接続
    • 同じ（X=8, Y=11）の点にRgの上端も接続
    → グリッドは真空管の「入力端子」。ここに来た信号が増幅されます

注意事項

⚠️ グリッド配線は高インピーダンスのため、配線は最短距離で行う

⚠️ Rgを忘れると動作しません！必ず取り付けてください

→ 長い配線はアンテナになってノイズを拾いやすくなります

【工程4】カソード回路の構築

🤔 なぜこの作業をするのか？

カソード回路は真空管の動作点を決める重要な部分です。Rkに電流が流れると電圧が発生し、これによりグリッドが相対的にマイナスになります（自己バイアス）。Ckは音声信号に対してはRkをショートさせ、ゲインを上げる役割があります。SW1でこのCkを切り離すことで、ゲインを調整できます。

作業内容

  1. Rk（1.5kΩ）の取付
    • 位置：（X=15, Y=14）から（X=15, Y=16）垂直配置
    • カラーコード確認：茶・緑・赤・金
    • 上端をPin3側、下端をGND側に配置
    → 1.5kΩは真空管に適切なバイアスを与える標準的な値です
  2. Ck（22μF電解コンデンサ）の取付
    • 位置：（X=17, Y=14）から（X=17, Y=16）Rkと並列
    • 極性注意：長い足（+）を上側（Pin3側）へ
    • 本体の白い帯（-マーク）が下側（GND側）になることを確認
    → 電解コンデンサは逆接続すると破裂する危険があります！
  3. Pin3からカソード部品への配線
    • 橙色の単線でPin3（X=15, Y=11）から（X=15, Y=14）へ
    • さらに（X=15, Y=14）から（X=17, Y=14）へ横配線でCkの+側にも接続
    → カソードから流れ出る電流がRkとCkの両方を通るようにします
  4. SW1用配線の準備
    • 青色のより線を15cm用意
    • （X=15, Y=16）のRk下端から引き出し
    → SW1でRkの下端をGNDに直結するか選択できるようにします

確認ポイント

✓ Ckの極性が正しいか再確認（逆接続は破損の原因）

✓ RkとCkが並列に接続されているか確認

→ 並列接続により、DC的にはRkが働き、AC的にはCkがショートとして働きます

【工程5】プレート回路とSW2配線

🤔 なぜこの作業をするのか？

プレート抵抗（Ra）は、真空管の出力特性を決める重要な部品です。抵抗値が大きいほど出力電圧は大きくなりますが、同時に歪みやすくなります。100kΩではクリーンな音、470kΩでは歪みやすい音になります。これは、大きな抵抗では電圧降下が大きくなり、信号の振幅が電源電圧に近づくと波形がクリップ（頭打ち）するためです。

作業内容

  1. Ra1（100kΩ）の取付
    • 位置：（X=20, Y=5）から（X=23, Y=5）水平配置
    • カラーコード：茶・黒・黄・金
    → クリーンサウンド用の標準的な値
  2. Ra2（470kΩ）の取付
    • 位置：（X=20, Y=3）から（X=23, Y=3）水平配置
    • カラーコード：黄・紫・黄・金
    → オーバードライブサウンド用の高い値
  3. プレート抵抗の電源接続
    • 赤色単線で（X=23, Y=5）から（X=23, Y=2）の+9Vラインへ
    • 同様に（X=23, Y=3）から（X=23, Y=2）へ接続
    → どちらの抵抗も+9V電源に接続し、SW2で選択します
  4. SW2配線の作成
    • Pin1（X=13, Y=9）から灰色より線でSW2中央端子へ（20cm）
    • SW2左端子から（X=20, Y=5）Ra1左端へ（15cm）
    • SW2右端子から（X=20, Y=3）Ra2左端へ（15cm）
    → SW2は「どちらの抵抗を使うか」を選ぶスイッチです

注意事項

⚠️ SW2は3端子のSPDTスイッチを使用。中央が共通端子

→ 2つの抵抗を同時に接続すると、並列合成抵抗になって音が変わります

【工程6】出力部の配線

🤔 なぜこの作業をするのか？

C2（出力カップリングコンデンサ）は、増幅された信号から直流成分を取り除きます。真空管のプレートには高い直流電圧がかかっているので、これをカットしないと次の機器を壊してしまいます。0.47μFという大きめの値は、低音もしっかり通すためです。ボリュームポットは可変抵抗器で、出力信号をどれだけGNDに逃がすかで音量を調整します。

作業内容

  1. C2（0.47μF）の取付
    • 位置：（X=22, Y=9）から（X=22, Y=11）
    • フィルムコンデンサまたはオーディオグレード推奨
    → C1より大きい値で、ベースやミッドレンジをしっかり伝えます
  2. Pin1からC2への配線
    • 橙色単線で（X=13, Y=9）Pin1から（X=22, Y=9）C2入力へ
    • プレート信号を出力コンデンサへ伝送
    → Pin1（プレート）は増幅された信号の出力端子です
  3. ボリュームポットへの配線
    • 黄色より線を25cm用意
    • （X=22, Y=11）C2出力からVolumeポット中央端子へ
    • 黒色線でVolumeポットの端子3（GND側）をGNDラインへ
    → ポットは信号を端子1-3間の抵抗で分圧し、端子2から取り出します
  4. 出力ジャックへの配線
    • 黄色より線でVolumeポット端子2から出力ジャックTipへ
    → ボリュームで調整された信号が最終出力となります

【工程7】ヒーター配線

🤔 なぜこの作業をするのか？

ヒーターは真空管内部のフィラメントで、カソードを加熱して電子を放出しやすくします。通常12AX7は12.6V/0.3Aで動作しますが、9Vでも薄暗く光り、動作します。これは「スターブド・プレート（飢餓電圧）」動作と呼ばれ、独特の歪みキャラクターを生み出します。ヒーター配線は交流ノイズの原因になりやすいので、他の信号線から離して配線します。

作業内容

  1. Pin4（ヒーター1）の配線
    • 緑色単線で（X=17, Y=9）Pin4から（X=17, Y=2）+9Vラインへ
    • 垂直に配線して他の信号線との交差を最小化
    → 垂直配線により、横に走る信号線との電磁的干渉を減らします
  2. Pin5（ヒーター2）の配線
    • 緑色単線で（X=17, Y=11）Pin5から（X=17, Y=18）GNDラインへ
    • こちらも垂直配線を心がける
    → Pin4とPin5の間にヒーターがあり、電流が流れて発熱します

確認ポイント

✓ Pin4-Pin5間の抵抗値を測定（12AX7の場合、冷時で約10-20Ω）

→ この抵抗値はヒーターが正常であることを示します

【工程8】電源部の構築

🤔 なぜこの作業をするのか？

Cin（電源フィルタコンデンサ）は、電源のリップル（脈動）やノイズを除去する重要な部品です。電源アダプターからの電圧は完全な直流ではなく、わずかに波打っています。この220μFという大容量のコンデンサが「貯水池」のように働き、電圧を平滑化します。これがないと、電源ノイズが増幅されて「ブーン」というハム音になります。

作業内容

  1. Cin（220μF）の取付
    • 位置：（X=25, Y=15）+側、（X=25, Y=17）-側
    • 極性確認：+マークまたは長い足を上側（+9V側）へ
    • 耐圧25V以上のものを使用
    → 耐圧は使用電圧の2倍以上が安全です（9V×2＝18V以上）
  2. 電源接続
    • 赤色単線で（X=25, Y=15）から（X=25, Y=2）+9Vラインへ
    • 黒色単線で（X=25, Y=17）から（X=25, Y=18）GNDラインへ
    → 電源入力のすぐ近くに配置して、ノイズを元から断ちます
  3. DCジャックの配線
    • ⚠️ 重要：BOSSアダプターはセンターマイナス！
    • 黒色より線でDCジャック中心ピンからGNDラインへ
    • 赤色より線でDCジャック外側端子から+9Vラインへ
    • 極性：センターマイナス（一般的なエフェクター仕様）
    → 極性を間違えると即ショート！必ずテスターで確認してください

【工程9】外部配線の完成

🤔 なぜこの作業をするのか？

入出力ジャックとスイッチは、ユーザーが直接触れる部分です。ジャックのスリーブ（外側）をGNDに接続することで、シールド効果を持たせ、外部ノイズを防ぎます。SW1はCkをバイパスするかどうかの切り替えで、音のキャラクターを変えます。配線を整理することで、見た目だけでなく、ノイズ対策にもなります。

作業内容

  1. 入力ジャックの配線
    • Tip：白色線をC1入力側へ（既に準備済み）
    • Sleeve：黒色線でGNDラインへ
    → Tipが信号、Sleeveがグランド（シールド）です
  2. 出力ジャックの配線
    • Tip：Volumeポットからの黄色線
    • Sleeve：黒色線でGNDラインへ
    → 入力と同じく、確実なグランド接続が重要
  3. SW1の配線
    • 端子1：Rk下端からの青色線
    • 端子2：黒色線でGNDラインへ
    • スイッチON＝ショート（Ckバイパス無効）
    → ON時はRkのみ（クリーン）、OFF時はRk//Ck（高ゲイン）
  4. LED配線（新規追加）
    • 電源LED（赤）：
      • +9Vライン → LED赤アノード（長い足）
      • LED赤カソード → 1kΩ抵抗 → GND
      • ケース前面にベゼルで固定
    • OD表示LED（青）：
      • SW2のRa2側端子 → LED青アノード（長い足）
      • LED青カソード → 1kΩ抵抗 → GND
      • SW2がRa2選択時のみ点灯
    → LEDの極性に注意！長い足がアノード（+側）です
  5. 配線の整理とスターグランド
    • ケーブルタイで配線を束ねる
    • 信号線と電源線を分離
    • 熱収縮チューブで接続部を保護
    • スターグランド実装（ノイズ対策）：
      • 入力ジャックのスリーブを「GND集中点」とする
      • 出力ジャックスリーブ → 入力ジャックスリーブへ
      • 基板GNDライン → 入力ジャックスリーブへ
      • DCジャック（-）端子 → 入力ジャックスリーブへ
      • 金属ケースの場合 → 入力ジャックスリーブへ
    • シールド線の使用（推奨）：
      • 入力ジャック〜基板間に使用
      • シールドは片端接地（ジャック側のみGND接続）
      • 両端接地はグランドループの原因になるので避ける
    → スターグランドによりハムノイズを大幅に低減できます

【工程10】最終確認と動作テスト

🤔 なぜこの作業をするのか？

電子回路は小さなミスが大きな故障につながります。段階的なテストにより、問題を早期発見し、高価な真空管や接続機器を守ります。特に初回電源投入時は慎重に行い、異常があればすぐに電源を切ることが重要です。

電源投入前の確認

  1. DCジャック極性確認（最重要）
    • テスターでDCジャックの極性を確認
    • 中心端子：マイナス（GND）
    • 外側端子：プラス（+9V）
    • BOSSアダプターはセンターマイナス！
    → 極性を間違えると即座に回路が破損する可能性があります
  2. 目視検査
    • はんだ付け部分にイモはんだやブリッジがないか
    • 部品の向き（特に電解コンデンサ）が正しいか
    • 配線の色が設計通りか
    → 目で見て分かる問題は、この段階で必ず修正します
  3. 導通チェック
    • +9VとGND間：開放（∞Ω）
    • 入力Tip→C1→Pin2の導通
    • Pin1→C2→Volume→出力Tipの導通
    • 各スイッチの動作確認
    → 信号の流れる経路が正しくつながっているか確認
  4. 抵抗値測定
    • Pin4-Pin5間：10-20Ω（ヒーター）
    • SW2切替時のPin1-+9V間：100kΩ/470kΩ
    → 部品が正しく接続されていることの最終確認

動作テスト

  1. アダプター単体の確認
    • アダプターをコンセントに接続（本体には未接続）
    • テスターでプラグの電圧と極性を確認
    • 中心：0V（GND）、外側：+9V
    → まずアダプター自体が正常か確認
  2. 真空管を挿さずに電源ON
    • Pin4-Pin5間に9V来ているか確認
    • 異常な発熱がないか確認
    • 電源LED（赤）が点灯することを確認
    → まず真空管なしで基本動作を確認します
  3. 電源OFFして真空管を挿入
    • 向きに注意（切り欠きを合わせる）
    • まっすぐ押し込む
    → 斜めに挿すとピンが曲がります
  4. 再度電源ON
    • ヒーターがオレンジ色に光るか確認（薄暗い光が正常）
    • Volume最小でアンプに接続
    • ギターを接続して音出し確認
    → 9Vでは薄暗い光ですが、これが正常です
  5. 機能テスト
    • SW1：ON/OFFでゲイン変化確認
    • SW2：クリーン/ODの切替確認（青LED連動）
    • Volume：音量調整の確認
    → すべての機能が設計通り動作することを確認

トラブルシューティング

  • 音が出ない：信号経路を入力から順に追跡
    → オシロスコープやオーディオプローブで信号を追跡
  • ノイズが多い：シールド処理、グランドループ確認
    → すべてのGNDが一点に集まっているか確認
  • 歪みすぎる：Ra値の確認、電源電圧確認
    → 部品の値が間違っていないか再確認
  • 発振する：配線の引き回し、入出力の分離
    → 高ゲイン時は配線がアンテナになりやすい
